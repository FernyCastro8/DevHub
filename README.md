### Installation

- Open your favorite text editor of choice
- cd to directory then run:
`` npm install ``
- cd to client then run:
`` npm install ``
- cd to server then run:
`` npm install ``

and enter the following command:
   ``npm run dev`` to run it locally at localhost:3000.
